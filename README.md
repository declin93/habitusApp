# Habitus - Habit Tracker PWA

A beautiful habit tracker with GitHub-style contribution matrix, built with React + Vite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Update `base` in `vite.config.js` to match your repo name
2. Run: `npm run build && npm run deploy`

## Features

- ✅ Dark/light theme
- ✅ Category filters
- ✅ Dashboard view
- ✅ Drag-to-reorder habits
- ✅ Active days per week
- ✅ Weekly completion targets
- ✅ Streak milestones & badges
- ✅ Streak freeze mechanic
- ✅ Journal notes per day
- ✅ Multi-check-in mode
- ✅ Multiple daily reminders
- ✅ Sound feedback
- ✅ Export/import data
- ✅ PWA (offline support)
- ✅ Fully responsive mobile UI
